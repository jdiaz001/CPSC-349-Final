TextBookTrade

Description

TextbookTrade is a full-stack web application that allows university students to buy, sell, and trade textbooks in a streamlined and user-friendly environment. The platform features secure user authentication, listing management with image uploads, and a built-in messaging system for direct communication between users.

Challenges

  1. Challenge: Allowing users to upload and view textbook images securely without writing them to disk.

    Solution: Stored image data in MySQL using a BLOB field. Used base64 encoding to convert image binary data into a format embeddable in HTML. Created a @app.template_filter('b64encode') Jinja2 filter to render images in templates.

  2. Challenge: Enabling private messaging between users with proper access control.

    Solution: Built a messaging model and message route that checks login status. Displayed messages on the profile page, and created a dedicated message view page for reading individual messages.

  3. Challenge: Initially, the textbook table didn’t support multiple images, making it hard to associate several pictures with one textbook.

    Solution: Introduced a separate TextBookPictures table with a foreign key to the textbook ID. This allowed multiple images per listing and more flexible image management.

Outcomes

  1. Real-Time Messaging Between Users.
  2. Dynamic Image Handling.
  3. Smooth User Experience.
  4. Seamless Textbook Listing System.
  5. Full-Featured User Authentication.
